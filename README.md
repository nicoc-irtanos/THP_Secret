Ruby version : 2.5.1

Rails version : 5.2.0

## Utilisation en local

```git clone https://github.com/nicolasculerier/THP_Secret```

```cd THP_Secret```

```bundle install --without production```

```rails db:migrate```

```rails db:seed``` (pas forcément ultra utile comme tu peux créer un utilisateur direct sur le site)

```rails server```

Voilà allez peace !
